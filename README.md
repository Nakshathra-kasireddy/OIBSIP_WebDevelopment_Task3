# 🌡️ Temperature Converter Web App

## 📌 Task Overview
This project is a simple **Temperature Converter Web Application** built using HTML, CSS, and JavaScript.  
It allows users to convert temperature values between Celsius and Fahrenheit, with an optional extension to Kelvin.

---

## 🎯 Objective
The main objective of this project is to:
- Build an interactive web application using JavaScript
- Understand user input handling
- Perform real-time temperature conversion
- Practice DOM manipulation
- Improve frontend development skills

---

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- VS Code (Editor)

---

## 📚 Features
- Input field to enter temperature value
- Dropdown / radio buttons to select temperature unit (Celsius or Fahrenheit)
- Convert button to perform conversion
- Display area for converted result
- Input validation to ensure only numbers are entered
- Optional feature: Conversion to Kelvin (bonus)

---

## 🔄 How It Works
1. User enters a temperature value
2. User selects the input unit (Celsius or Fahrenheit)
3. User clicks the "Convert" button
4. The app calculates and displays the converted temperature with the correct unit
